|        文件名        |                    功能描述                    |
| :------------------: | :--------------------------------------------: |
| `decompose_agent.py` | 使用 `qwen-2.5` 模型将原始问题分解为多个子问题 |
|   `text_agent.py`    |  接收子问题 + 文本内容，尝试回答或输出不确定   |
|   `image_agent.py`   | 接收子问题 + 图像内容，使用 `qwen-vl-2.5` 回答 |
|  `summary_agent.py`  |      接收所有子问题及答案，总结出最终答案      |
